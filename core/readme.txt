sip-servlets-weld core project. Provides the bootstrap functionality and basic sip-servlets CDI tools for the following containers:

1. MSS 1.5 & MSS 1.6
	* Tomcat 6.x
	* JBoss AS5
2. MSS 2.x 
	* Tomcat 7.x
	
In order to build, in the top core/ directory run:
mvn clean compile install

** Use Maven 2.x to build
